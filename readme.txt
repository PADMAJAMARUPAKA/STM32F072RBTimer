******Description********	
  - This is a basic project which uses the Timer3 in upcounting mode.
  - An interrupt is generated at overflow to toggle LED4.
   - In this example, SysClock is configured at 48 MHz
*****Hardware and Software environment***
  - This example runs on STM32F072RB devices.
  - Keil toolchain is the toolchain used.
****How to use it****
In order to make the program work, you must do the following :
 - Open your preferred toolchain
 - Rebuild all files and load your image into target memory
 - Run the example
